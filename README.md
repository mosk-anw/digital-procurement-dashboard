# Digital Procurement Manager Dashboard

An interactive dashboard showcasing the AI-powered procurement platform for food & beverage businesses.

## Features

- **ROI Calculator**: Calculate potential savings and efficiency gains
- **Interactive Dashboard**: Toggle between Value Proposition and Technical Architecture views
- **Real-time Analytics**: Visualizations of procurement improvements
- **Responsive Design**: Works on desktop, tablet, and mobile

## View Online

Visit the live dashboard: [Digital Procurement Manager](https://yourgithub.github.io/digital-procurement-dashboard/)

## Quick Start

Simply open `index.html` in your browser to view the dashboard locally.

## Technology Stack

- **Frontend**: Tailwind CSS, Chart.js
- **Hosting**: GitHub Pages
- **Framework**: Vanilla JavaScript (no build tools required)
